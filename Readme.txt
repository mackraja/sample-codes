How to run this sample-codes Task :

Requirement :

1.) PHP 5.X
2.) mod_rewrite module must be enable.

How to run :

1.) (Windows) just copy this folder to root / (Unix) also Give permissions 

Features :

1.) You can change Email, Extension, Folder name of Views files.
2.) You can change layout.
3.) You can add Elements.
4.) You can add header and footer.
5.) You can set Partial.
6.) You can directly access any page. Just write name of the page in link. See Screen-shot folder
Example : 1.) http://localhost/sample-codes/service
	  2.) http://localhost/sample-codes/carrer
